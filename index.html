HTML & CSS cheatsheet

HTML Tags

Tag Description Example
<html> Defines the root of the HTML document <html> Your HTML content here </html>
<head> Contains meta information about the document/website <head>Meta information inside like title</head>
<title> Defines the title for the document/websites, used inside <head> <title> My Page Name </title>
<body> Contains the content of the document <body> Content of the page </body>
<h1> to <h6> It's used for headers with diffent sizes. H1 biggest to H6 smallest <h1> Welcome to my page </h1>
<p> Defines a paragraph <p> This is a paragraph </p>
<a> Defines a hyperlink <a href="https://www.example.com"> Visit this link </a>
<img> Embeds an image. <img src="image.jpeg" alt="Description of the image(not visible)">
<div> Defines a division or a section in the HTML document <div> You can place paragraphs and other stuff inside </div>
<span> Defines a section in a document, can be used inside other tags like <p> <span> Can change a section of a paragraph for example</span>
<ul>, <ol>, <li> Defines unordered an ordered lists and list items <ul> <li> Item 1 </li> <li> Item 2 </li> </ul>
<table>, <tr>, <td>, <th> Defines table, table row, table data and table header

<table>

<tr>

<th>Name</th>

</tr>

<tr>

<td> John <td>

</tr>

</table>
<form>, <input>, <button> Defines an input form and its elements

<form action="/submit" method="post">

<input type="text" name="username" placeholder="Enter your usernmame">

<button type="submit">Submit</button>

</form>
<iframe> Embeds a webpage within a webpage <iframe src="https://www.example.com"></iframe>
